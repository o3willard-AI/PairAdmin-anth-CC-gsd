    **Create `frontend/src/components/terminal/TerminalPreview.tsx`:**

    Props: `tabId: string`

    Implementation using `useRef` + `useEffect` pattern (per locked decision):
    ```typescript
    import { useEffect, useRef } from "react";
    import { Terminal } from "@xterm/xterm";
    import { FitAddon } from "@xterm/addon-fit";
    import { CanvasAddon } from "@xterm/addon-canvas";
    import "@xterm/xterm/css/xterm.css";
    ```

    In the useEffect (dependency: `[tabId]`):
    1. Guard: `if (!containerRef.current) return;`
    2. Create `new Terminal({...})` with theme: background `#0d0d0d`, foreground `#d4d4d4`, cursor `#d4d4d4`, fontSize 13, fontFamily `"'JetBrains Mono', 'Fira Code', monospace"`, scrollback 1000, convertEol true.
    3. Create `new FitAddon()` and load it via `term.loadAddon(fitAddon)`.
    4. Call `term.open(containerRef.current)`.
    5. **AFTER open()**: Create `new CanvasAddon()` and load via `term.loadAddon(canvasAddon)`. CRITICAL: CanvasAddon must be loaded AFTER open() or it silently fails.
    6. Call `fitAddon.fit()`.
    7. Write mock terminal content that simulates a realistic bash session:
       ```
       term.writeln("\x1b[32m$ \x1b[0mls -la /home/admin");
       term.writeln("total 48");
       term.writeln("drwxr-xr-x  6 admin admin 4096 Mar 26 09:12 .");
       term.writeln("drwxr-xr-x  3 root  root  4096 Mar 20 14:30 ..");
       term.writeln("-rw-r--r--  1 admin admin  220 Mar 20 14:30 .bash_logout");
       term.writeln("-rw-r--r--  1 admin admin 3771 Mar 20 14:30 .bashrc");
       term.writeln("\x1b[32m$ \x1b[0mgit status");
       term.writeln("On branch main");
       term.writeln("nothing to commit, working tree clean");
       term.writeln("");
       term.writeln("\x1b[33m[No terminal connected — Phase 1 mock]\x1b[0m");
       ```
    8. Store terminal in `termRef.current = term`.
    9. Create a `ResizeObserver` on `containerRef.current` that calls `fitAddon.fit()` on resize.
    10. Cleanup function: `resizeObserver.disconnect()`, `term.dispose()`.

    Wrap CanvasAddon loading in a try/catch — if it fails (unlikely with canvas, but safe practice), log a warning and continue without acceleration. This is NOT an error condition.

    Render: `<div ref={containerRef} className="h-full w-full overflow-hidden" style={{ minHeight: "120px" }} />`
