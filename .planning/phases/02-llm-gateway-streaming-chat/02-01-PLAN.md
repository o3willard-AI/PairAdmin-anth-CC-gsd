    Create `services/llm/provider.go`:
    ```go
    package llm

    import "context"

    type Role string
    const (
        RoleSystem    Role = "system"
        RoleUser      Role = "user"
        RoleAssistant Role = "assistant"
    )

    type Message struct {
        Role    Role
        Content string
    }

    type StreamChunk struct {
        Text  string
        Done  bool
        Error error
    }

    type Usage struct {
        InputTokens  int
        OutputTokens int
    }

    type Provider interface {
        Name() string
        Stream(ctx context.Context, messages []Message) (<-chan StreamChunk, error)
        TestConnection(ctx context.Context) error
    }
    ```

    Create `services/llm/registry.go` — simple map-based registry:
    - `Registry` struct with `providers map[string]Provider`
    - `Register(p Provider)`, `Get(name string) (Provider, error)` methods

    Create `services/llm/openai.go` — `OpenAIProvider` struct:
    - Fields: `apiKey string`, `baseURL string` (empty = OpenAI default), `model string`
    - `NewOpenAIProvider(apiKey, baseURL, model string) *OpenAIProvider`
    - `Stream()`: use `github.com/openai/openai-go/v3` client; create with `option.WithAPIKey(apiKey)` and `option.WithBaseURL(baseURL)` if baseURL != ""; call `client.Chat.Completions.NewStreaming(ctx, params)`; read from accumulator; send chunks to buffered channel (size 32); close channel on done
    - `TestConnection()`: call `client.Models.List(ctx)` — returns error on auth failure
    - OpenRouter: same struct, pass `BaseURL="https://openrouter.ai/api/v1"` and `OPENROUTER_API_KEY`
    - LM Studio: same struct, pass `BaseURL="http://localhost:1234/v1"`, empty apiKey

    Create `services/llm/anthropic.go` — `AnthropicProvider` struct:
    - `NewAnthropicProvider(apiKey, model string) *AnthropicProvider`
    - `buildParams(messages []Message) anthropic.MessageNewParams`: loop messages; when Role==RoleSystem, set `params.System = []anthropic.TextBlockParam{{Text: m.Content}}`; other roles go into params.Messages array
    - `Stream()`: call `client.Messages.NewStreaming(ctx, params)`; iterate via `stream.Next()` / `stream.Current()`; emit text delta chunks; send Usage on done
    - `TestConnection()`: list models or make a minimal completion — any non-auth error is acceptable

    Create `services/llm/ollama.go` — `OllamaProvider` struct:
    - `validateOllamaHost(host string) error`: exported function; if host=="" return nil; parse URL; reject if hostname not in ["localhost","127.0.0.1","::1"]
    - `NewOllamaProvider(host, model string) (*OllamaProvider, error)`: calls `validateOllamaHost`; creates `api.NewClient` with resolved host URL
    - `Stream()`: wrap `client.Chat(ctx, req, callback)` in goroutine; callback sends to buffered channel (size 32); close channel when `resp.Done`

    Create `services/llm/context.go`:
    - `BuildMessages(systemPrompt, terminalContext, userInput string) []Message`: assembles [system, user] message slice; if terminalContext != "" prepend as fenced code block "```terminal\n{ctx}\n```\n\n" before userInput
    - `EstimateTokens(text string) int`: return `int(math.Ceil(float64(len(text)) / 4.0))`

    Create `services/llm_service.go` — `LLMService` struct:
    - `Config` struct: `Provider string`, `Model string`, `OpenAIKey string`, `AnthropicKey string`, `OpenRouterKey string`, `OllamaHost string`
    - `LoadConfig() Config`: reads all env vars (`PAIRADMIN_PROVIDER`, `PAIRADMIN_MODEL`, `OPENAI_API_KEY`, `ANTHROPIC_API_KEY`, `OPENROUTER_API_KEY`, `OLLAMA_HOST`)
    - `LLMService` struct: `ctx context.Context`, `cfg Config`, `activeProvider llm.Provider`
    - `NewLLMService(cfg Config) *LLMService`: creates appropriate provider based on cfg.Provider; stores as activeProvider
    - `Startup(ctx context.Context)`: saves ctx
    - `SendMessage(tabId, userInput, terminalContext string) error`: builds messages via `llm.BuildMessages`; launches goroutine that streams and emits `llm:chunk`, `llm:done`, `llm:error`, `llm:usage` events following exactly the 50ms batching pattern from RESEARCH.md Pattern 2
