    **Package-level injectable var (per D-02, matching commands.go lookPath pattern):**

    **Types:**
    ```go
    type PaneRef struct {
        ID   string // stable pane ID e.g. "%3" (per D-02)
        Name string // display name e.g. "main:0.0" (per D-01)
    }

    type TerminalUpdateEvent struct {
        PaneID  string `json:"paneId"`
        Content string `json:"content"`
    }

    type TerminalTabsEvent struct {
        Tabs []TabInfo `json:"tabs"`
    }

    type TabInfo struct {
        ID   string `json:"id"`
        Name string `json:"name"`
    }

    type captureState struct {
        lastHash uint64
    }

    type TerminalService struct {
        ctx       context.Context
        cancel    context.CancelFunc
        mu        sync.Mutex
        panes     map[string]*captureState // keyed by pane ID
        sem       *semaphore.Weighted
        emitFn    func(ctx context.Context, eventName string, optionalData ...interface{}) // injectable for testing
    }
    ```

    **Constructor:**
    ```go
    func NewTerminalService() *TerminalService {
        return &TerminalService{
            panes:  make(map[string]*captureState),
            sem:    semaphore.NewWeighted(4),
            emitFn: runtime.EventsEmit,
        }
    }
    ```

    **Startup** (called from main.go OnStartup):
    ```go
    func (t *TerminalService) Startup(ctx context.Context) {
        t.ctx, t.cancel = context.WithCancel(ctx)
        go t.pollLoop()
    }
    ```

    **pollLoop:**
    ```go
    func (t *TerminalService) pollLoop() {
        ticker := time.NewTicker(500 * time.Millisecond)
        defer ticker.Stop()
        for {
            select {
            case <-t.ctx.Done():
                return
            case <-ticker.C:
                t.tick()
            }
        }
    }
    ```

    **listPanes** (unexported):
    - Run `execCommand(ctx, "tmux", "list-panes", "-a", "-F", "#{pane_id}\t#{session_name}\t#{window_index}\t#{pane_index}")`
    - If error contains "no server running" or "error connecting to", return `nil, nil` (per D-03/D-04 — no crash when tmux absent)
    - Parse tab-delimited output: fields[0]=paneID, display name = fields[1]+":"+fields[2]+"."+fields[3]
    - Return `[]PaneRef`

    **capturePane** (unexported):
    - Run `execCommand(ctx, "tmux", "capture-pane", "-p", "-t", paneID)` — no `-e` flag (plain text, per anti-pattern in RESEARCH.md), no `-S` flag (visible screen only)
    - Trim trailing newlines: `strings.TrimRight(string(out), "\n")`
    - Apply filter pipeline before returning: `filter.NewPipeline(filter.NewANSIFilter(), credFilter).Apply(content)` — ANSI stripping is safety net even without `-e`, credential redaction protects content sent to frontend/LLM
    - Return filtered string

    **hashContent** (unexported):
    ```go
    func hashContent(s string) uint64 {
        h := fnv.New64a()
        h.Write([]byte(s))
        return h.Sum64()
    }
    ```

    **tick** (unexported, called every 500ms):
    1. Call `listPanes(t.ctx)` to get current panes
    2. Compute sets: `newPanes` (in current but not in `t.panes`), `removedPanes` (in `t.panes` but not in current)
    3. If membership changed (new or removed panes exist):
       - Add new panes to `t.panes` map with zero-value `captureState`
       - Remove removed panes from `t.panes` map
       - Build `TerminalTabsEvent` with ALL current panes and emit via `t.emitFn(t.ctx, "terminal:tabs", event)`
    4. Capture all current panes concurrently with semaphore (max 4):
       - Use `sync.WaitGroup` + `semaphore.Acquire/Release`
       - Collect results into a slice (protected by mutex or channel)
    5. After `wg.Wait()`, iterate results and for each pane:
       - Compute `hashContent(content)`
       - If hash differs from `t.panes[paneID].lastHash`, update hash and emit `t.emitFn(t.ctx, "terminal:update", TerminalUpdateEvent{PaneID: paneID, Content: content})`
    6. Emit from main goroutine AFTER wg.Wait() — not from capture goroutines (avoids EventsEmit thread safety question per RESEARCH.md)

    **Tests in `services/terminal_test.go`:**
    - Use `package services` (not `services_test`) to access unexported functions
    - Override `execCommand` in each test, restore in `defer`
    - For tests needing event emission verification: inject a mock `emitFn` that records calls into a slice
    - Test helper `fakeExecCommand` that returns a function producing an `*exec.Cmd` with canned stdout/stderr
    - Use `exec.CommandContext(ctx, "echo", output)` pattern for mock — or use `testing/iotest` approach where mock function sets stdout directly
    - For semaphore test: inject an execCommand that blocks on a channel; count concurrent goroutines via atomic counter; assert max <= 4
