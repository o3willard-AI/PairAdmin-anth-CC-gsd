type AuditLogger struct {

}

**NewAuditLogger(logDir string) (*AuditLogger, error):**
1. `os.MkdirAll(logDir, 0700)` — create log directory with restricted permissions
2. Compute filename: `filepath.Join(logDir, fmt.Sprintf("audit-%s.jsonl", time.Now().Format("2006-01-02")))`
3. Create `&lumberjack.Logger{Filename: filename, MaxSize: 100, MaxAge: 30, Compress: false}`
4. Create `slog.NewJSONHandler(rotator, &slog.HandlerOptions{Level: slog.LevelInfo})`
5. Return `&AuditLogger{logger: slog.New(handler)}, nil`

**Write(entry AuditEntry) error** method on `*AuditLogger`:
1. Nil-safe check: `if a == nil || a.logger == nil { return nil }`
2. Call `a.logger.Info("audit", slog.String("event", entry.Event), slog.String("session_id", entry.SessionID), slog.String("terminal_id", entry.TerminalID), slog.String("content", entry.Content))`
3. Return `nil`

Per D-03: MaxSize=100MB, MaxAge=30 days, Compress=false.
